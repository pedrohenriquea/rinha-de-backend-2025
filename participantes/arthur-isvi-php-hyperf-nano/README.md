
# 🚀 Rinha de Backend 2025

## 🛠️ Tecnologias

### Backend
- **Web Api**: **PHP** com **Hyperf/Nano Framework**
- **Health-Check Worker**: **PHP** + **Swoole**
- **Payments Worker**: **PHP** + **Swoole**
- **Database/Cache/Queue**: **Redis**

### Infraestrutura
- **Containerização**: **Docker**
- **Orquestração**: **Docker-compose**
- **Load Balancer**: **Nginx**
